About The Project

This project is part of [CSP203] ie.. Software System laboratory course at Indian Institiute of Tecnology Ropar (IIT-RPR). In this course all students are divided into groups of four students per group. Each group has to build one project and maintain a website for the same.

This course is being taught by Dr. Mukesh Saini and Dr. Neerja Goel (assistant professor at IIT Ropar). Students are supposed to propose a project that could be implemented in C/C++/Java, describing its end goal, target audience, features, shortcomings/limitations, etc. Traditional ideas should be productized well.
